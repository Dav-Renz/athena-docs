# Weapon

### weapon

Used to get a weapon and add to self.

{% tabs %}
{% tab title="Usage" %}
```
/weapon [weapon_name]
```
{% endtab %}

{% tab title="Example" %}
```
/weapon smg
```
{% endtab %}
{% endtabs %}

### removeallweapons

Remove all weapons from self.

{% tabs %}
{% tab title="Usage" %}
```
/removeallweapons
```
{% endtab %}
{% endtabs %}
