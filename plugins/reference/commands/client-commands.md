# Client Commands

These commands are ran on client-side by typing them into the `f8` console menu.

### pos

Get current position. Print to console.

Alternative: `/position`

{% tabs %}
{% tab title="Usage" %}
```
/pos
```
{% endtab %}
{% endtabs %}

### heading

Get current heading. Print to console.

{% tabs %}
{% tab title="Usage" %}
```
/heading
```
{% endtab %}
{% endtabs %}

### id

Get own identifier.

{% tabs %}
{% tab title="Usage" %}
```
/id
```
{% endtab %}
{% endtabs %}
